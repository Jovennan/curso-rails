O QUE É RAILS?

Convention Over Configuration

"Rails is the killer app for Ruby." - Matz

Don't Repeat Yourself

Rails tem opiniões sobre a melhor maneira de fazer muitas coisas em um aplicativo da web, e predefinições para este conjunto de convenções, ao invés de exigir que você especifique cada minúcias através de arquivos de configuração sem fim. Rails é uma estrutura de desenvolvimento de aplicações web escrita na linguagem Ruby. Ele é projetado para tornar mais fácil a programação de aplicativos da Web fazendo suposições sobre o que todo desenvolvedor precisa para começar. DRY é um princípio de desenvolvimento de software que afirma que "todo pedaço de conhecimento deve ter uma única, inequívoca, representação autoritária dentro de um sistema". Ao não escrever as mesmas informações uma e outra vez, nosso código é mais sustentável, mais extensível e menos bugado.
